# Roofing Ops Dashboard

Operational dashboard concept for roofing teams that need project status, material flow, office follow-up, and field visibility in one place.

## Problem

Roofing operations often depend on scattered spreadsheets, PDFs, messages, and job folders. This project explores a cleaner interface for tracking daily work without hiding important business details.

## Planned Features

- Project overview cards
- Work order status table
- Material request queue
- Office follow-up list
- Field update timeline
- Light/dark theme support

## Tech Stack

- React
- React Query
- Ant Design
- AG Grid
